<objective>
Create `test/cleanup-deployment.sh` — a standalone teardown script that an operator runs after inspecting a hello-world E2E deployment to remove every resource the test created.

Purpose: Complete the provision -> verify -> teardown loop required for v1. Without this, operators who run the E2E test must manually delete five different resources (staging app, production app, Coolify project, two Docker volumes, Doppler project) via the Coolify UI, SSH, and the Doppler dashboard. CLEAN-01 + CLEAN-02 close that gap with a single command driven by the JSON report file produced by Phase 2's E2E test.

Output: A single new file `test/cleanup-deployment.sh` (~100 lines) that reads a `test/results/YYYYMMDD-HHMMSS.json` report file, extracts all credentials and UUIDs, and executes the same five-step deletion sequence used in `test/e2e.sh` `cleanup()` (lines 109-148).
</objective>
